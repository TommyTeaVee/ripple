Ripple é um "botão de pânico" que através de um gatilho envia a sua mensagem para todos os aplicativos que forem configurados para responder esta emergência. Tais aplicativos podem fazer coisas como trancar o telefone, disfarçarem-se, excluir os dados privados, enviar uma mensagem de emergência e muito mais. Foi feito no intuito de serem usados em situações onde haja tempo para reagir sem que acionem estas ações por engano.

Esta é uma versão BETA do aplicativo! Atualmente nós estamos trabalhando para darmos suporte ao máximo de aplicativos possíveis. Já é compatível com ChatSecure, Orweb, Umbrella e Zom. Os próximos serão: Courier, PanicButton, OpenKeychain, Orfox, SMSSecure, StoryMaker.

Aqui dois cenários hipotéticos:

• Uma organização é invadida regularmente em suas instalações por forças de segurança que procuram em todos os computadores e dispositivos móveis. A organização sabe com antecedência que uma invasão irá acontecer num prazo de um a dois minutos. Eles precisam de uma maneira bastante confiável para apagar todos os dados sensíveis dos aplicativos.

• Um trabalhador humanitário possuí diversos dados confidenciais das pessoas no seu dispositivo.  Regularmente sincroniza estes dados com segurança em um banco de dados central.  Ocasionalmente, o trabalhador humanitário precisa deixar o país em um curto espaço de tempo.  Regularmente os guardas de fronteira descarregam todo o conteúdo dos dispositivos móveis das pessoas que ali atravessam.  Enquanto aguarda na fila, o trabalhador humanitário vê os guardas apreendendo os dispositivos das pessoas e então se lembra de todos os dados no dispositivo, ela destrava o telefone e aciona o gatilho de limpeza que exclui todas os aplicativos confidenciais do dispositivo. Quando este trabalhador retorna ao escritório central, o seu dispositivo é novamente sincronizado com o banco de dados da central.
